python-console
=====
This is the result of a small side project to write a Qt widget that
encapsulates an interactive Python shell.

Quickstart
-----
You should have Qt4 and Python libraries. You will need CMake to build this
project as follows:

1. mkdir build
2. cmake ..
3. make
